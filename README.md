# 擷取🐼的網頁菜單資訊

在chrome書籤列新增javascript腳本，點選後自動擷取🐼網頁的菜單至剪貼簿。其擷取格式為 `產品名稱` + `,` + `價格` ，可直接貼至 `Din.Ben.Don!` 新增店家。

## 參考擷取結果

``` csv
名間鄉冬片仔,30.00
二七仔烏龍,30.00
蜜香大冇紅茶,30.00
十八胭脂紅玉,30.00
```
